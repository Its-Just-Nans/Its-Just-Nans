## Shortcuts Normal mode

- `h`, `j`, `k`, `l`: move cursor
- `x`: delete character
- `ro`: replace current letter by `o`
- `dd`: delete and copy line

## Files

```vim
# open/create file in new tab
:new filename.ext

# open/create file in new vertical split
:vert new filename.ext
```

```vim
# switch between splits
CTRL-w h
CTRL-w j
CTRL-w k
CTRL-w l
```
