## Shortcuts

- `Magic` + `D`: open launcher
- `Magic` + `Enter`: open terminal
- `Magic` + `0-9`: switch workspace
- `Magic` + `Shift` + `0-9`: move window to workspace

- `Magic` + `e`: default layout
- `Magic` + `s`: stacking layout
- `Magic` + `w`: tabbed layout
- `Magic` + `f`: fullscreen
- `Magic` + `r`: resize mode

- `Magic` + `Shift` + `Space`: toggle floating
- `Magic` + `LeftClick`: drag floating
- `Magic` + `Shift` + `c`: restart in place
