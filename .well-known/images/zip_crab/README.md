# zip crab

This logo represents a crab holding a zip file, symbolizing the concept of compressed data.

## Convert to PNG

```sh
# example using resvg
cargo install resvg --locked
resvg zip_crab.svg zip_crab.png
resvg --background white zip_crab.svg zip_crab_white.png
```
