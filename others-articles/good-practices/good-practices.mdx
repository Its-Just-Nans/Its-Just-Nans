## Data

- Save your data, online and offline, on different disks

## Files

- sort your files
- use only a subset of ASCII char to name a file `A-Z`, `a-z`, `0-9`, `_`, `-`, `.`
- try to have only one dot in the filename
- if possible, date the file starting with `YYYY-MM-DD`

## Email

- Try to not forward mails (some mails contains direct link with autologin token)

## Website

- Do not depends on external ressources for your websites
